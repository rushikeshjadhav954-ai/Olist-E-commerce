# Olist-E-commerce
📌 Project Overview

This project analyzes the Brazilian e-commerce dataset from Olist to uncover customer behavior, sales trends, and delivery performance insights.
The analysis is performed using SQL, Excel, Power BI, and Tableau.


🧩 Data Model

The dataset follows a relational data model consisting of multiple interconnected tables.

  ## 🧭 Project Phases

### 1️⃣ Data Collection

* Imported Olist e-commerce dataset
* Reviewed multiple tables including orders, customers, products, payments, and reviews

### 2️⃣ Data Cleaning (SQL)

* Handled missing values and inconsistencies
* Standardized column formats (dates, text, IDs)
* Removed duplicates and irrelevant records

### 3️⃣ Data Transformation

* Performed joins between multiple tables
* Created derived columns (delivery time, weekday/weekend classification)
* Aggregated data for analysis

### 4️⃣ Exploratory Data Analysis (EDA)

* Analyzed customer behavior and purchase patterns
* Identified trends in sales, delivery, and reviews
* Compared weekday vs weekend transactions

### 5️⃣ Data Visualization

* Built interactive dashboards in Power BI and Tableau
* Visualized KPIs like sales trends, delivery time, and review scores
* Created filters for city, product category, and payment type

### 6️⃣ Insight Generation

* Derived key business insights from data
* Identified factors affecting customer satisfaction and delivery performance

### 7️⃣ Reporting & Presentation

* Created dashboards and project documentation
* Summarized findings in a clear and structured format for stakeholders

* ## 📌 Key Insights

* 📅 **Weekday vs Weekend Transactions**
  Weekday transactions dominate, contributing **~78% of total payments**, while weekends account for **~22%**, indicating customers prefer shopping during weekdays.

* ⭐ **Customer Satisfaction (Reviews)**
  A large share of customers provide high ratings, with approximately **61% of orders receiving 5-star reviews**, reflecting strong overall satisfaction.

* 🚚 **Delivery Performance**
  The average delivery time is around **12.5 days**, highlighting opportunities to improve logistics and delivery efficiency.

* 📍 **City-wise Purchasing Behavior (São Paulo)**
  Customers from São Paulo show higher purchasing activity, with payment values approximately **24% higher than the average product price**.

* ⏱️ **Delivery Time vs Review Score**
  Faster deliveries lead to higher review scores, while delayed deliveries significantly reduce customer satisfaction.
